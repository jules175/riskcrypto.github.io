## 📋 Fonctionnalités planifiées

### Marché & Données
- [ ] **Graphique de prix interactif** — courbe 7j / 30j / 90j avec zoom
- [ ] **Filtres avancés** — par catégorie (DeFi, Layer 1, Memecoins…)
- [ ] **Comparateur de cryptos** — mettre 2 actifs côte à côte
- [ ] **Top gainers / losers du jour** — section dédiée dans Marché
- [ ] **Dominance BTC en temps réel** — widget dans l'en-tête
- [ ] **Heatmap de marché** — visualisation couleur par performance

### Portfolio
- [ ] **Score de santé de portefeuille** — diversification, corrélation, concentration
- [ ] **Backtesting** — simuler les performances historiques sur 1 / 3 / 6 mois
- [ ] **Multi-portefeuilles** — gérer jusqu'à 5 portefeuilles distincts (Premium)
- [ ] **Export CSV** — télécharger ses transactions et P&L pour la fiscalité
- [ ] **Import depuis un exchange** — lire un CSV Binance / Coinbase

### Alertes & Signaux
- [ ] **Alertes on-chain** — gros transferts, mouvements de baleines
- [ ] **Alertes de volume inhabituel** — pic de volume détecté automatiquement
- [ ] **Signaux IA** — achat / vente basé sur RSI + sentiment + on-chain
- [ ] **Rapport hebdomadaire** — résumé du portefeuille envoyé chaque lundi

### Apprendre
- [ ] **Certification à la fin des 30 leçons** — badge partageable
- [ ] **Quiz hebdomadaire** — 5 questions sur les notions de la semaine
- [ ] **Mode révision** — revoir les leçons précédentes avec flashcards
- [ ] **Notion du jour dans l'onglet Apprendre** — pas seulement dans le menu

### Profil & Paramètres
- [ ] **Profil investisseur évolué** — questionnaire plus détaillé
- [ ] **Historique des recommandations** — voir les cryptos suggérées les jours précédents
- [ ] **Partage de profil** — partager son profil investisseur en image
- [ ] **Thème de couleur personnalisable** — accent color dans les paramètres

### Technique & Performance
- [ ] **Mode hors-ligne** — cache des dernières données pour consultation sans réseau
- [ ] **Widget iOS / Android** — prix et score affiché sur l'écran d'accueil
- [ ] **Haptic feedback** — retour tactile sur les interactions clés
- [ ] **Accessibilité** — labels VoiceOver / TalkBack sur tous les éléments
